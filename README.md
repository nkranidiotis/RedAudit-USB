# RedAudit ‚Äî Windows Host Forensics & Security Assessment Tool

RedAudit is a portable Windows host-forensics and security auditing framework designed for investigators, DFIR analysts, and security teams who need fast, structured, and defensible host assessment results.

The tool provides clear visibility into system activity, persistence mechanisms, external communications, and security posture ‚Äî all exported into a professional HTML report suitable for case files and technical documentation.

## üîç Features

### üß© Core Capabilities
- **Autoruns & Persistence Analysis**  
  Enumerates registry autoruns, system startup folders, scheduled tasks, and other persistence vectors.

- **External Network Connections**  
  Maps active TCP connections to process IDs, process names, and full executable paths.

- **System Integrity Checks**  
  Detects unquoted service paths, weak configurations, and missing patches.

- **Process & Service Overview**  
  Collects running processes and services with their respective metadata.

- **Risk-Based Findings & Scoring**  
  Assigns a host security score and produces prioritized findings with recommended actions.

- **Structured HTML Reporting**  
  Generates a clean, case-ready HTML report including findings, recommendations, and a detailed technical annex.
